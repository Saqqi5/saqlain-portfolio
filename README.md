# Saqlyn Digital Services Portfolio

A modern, responsive personal portfolio website for **Muhammad Saqlain Mushtaq** built with HTML, CSS, and JavaScript only.

## Features

- Premium mobile-first UI
- Dark and light mode
- Smooth reveal animations
- Services, portfolio, skills, and experience timeline sections
- Email-only contact section
- Download CV button
- SEO metadata, Open Graph card, robots.txt, and sitemap.xml
- GitHub Pages and Cloudflare Pages ready

## Deploy

### GitHub Pages

Deploy the repository root from the current branch in GitHub Pages settings.

### Cloudflare Pages

Use these settings:

- Framework preset: `None`
- Build command: leave empty
- Output directory: `/`
